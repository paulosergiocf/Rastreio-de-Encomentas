# RASTREAR ENCOMENDAS DO CORREIO
> Banco de Dados de Encomendas |
> Localiza encomendas, mostra datas, horários, locais e informações.

![1](https://user-images.githubusercontent.com/49497668/112072265-36533780-8b50-11eb-8578-5a700ba6d88e.png)

## Instalação

instalar módulo necessário:

```sh
 pip install pyrastreio
```
## Exemplo de Uso

Vejamos agora um exemplo de como utilizar a Ferramenta.

No terminal execute o script:

```sh
 python rastrear.pyw
```
Se estiver utilizando Windows basta dar dois cliques desde que o python esteja corretamente instalado.

Digite o código de rastreio e clique em "pesquisar", você verá uma lista organizada em tabela da movimentação de suas encomendas.

Você pode também adicionar o código de rastreio no menu cadastrar para não precisar digitar toda vez que for consultar o código de rastreio.
![2](https://user-images.githubusercontent.com/49497668/112072332-5be04100-8b50-11eb-8176-f2db18afd7d2.png)

Ao acessar o menu consultar verá que o item foi adicionado a tabela superior, dê dois cliques sobre o e o software efetuará a pesquisa do objeto.

![3](https://user-images.githubusercontent.com/49497668/112072258-33f0dd80-8b50-11eb-964f-0f961050c9b4.png)
